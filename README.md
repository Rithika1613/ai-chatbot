# ai-chatbotAI Chatbot using Generative AI
About the Project

This is a simple AI chatbot built using Python and Flask.
It uses a Generative AI API to answer user questions and have conversations in real time through a web page.

Features
AI-powered chatbot responses
Simple web interface
Real-time conversation
Secure API key using .env file
Technologies Used
Python
Flask
HTML, CSS
Generative AI API (Gemini/OpenAI)
dotenv
